# SwiftTUI Rendering Pipeline Applet

Independent static applet for exploring the SwiftTUI rendering pipeline.

Open `index.html` in a browser:

```bash
open applets/render-pipeline/index.html
```

The applet is intentionally self-contained and does not require Bun, npm, Astro,
or the public website package. It cites the checked-out SwiftTUI source paths
that define the pipeline and change-propagation flow.
